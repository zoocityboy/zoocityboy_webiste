---

## Prerequisites
- Dart SDK (version 2.12 or higher)
- Flutter (optional, but recommended)
- Git


## Getting Started

1. **Initialize Mason in your project**
   ```bash
   mason init
   ```

2. **Add a brick**
   ```bash
   mason add <brick_name>
   ```

3. **Get bricks**
   ```bash
   mason get
   ```
